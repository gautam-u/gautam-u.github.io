Support

Workbill is designed to work without account setup or onboarding friction. If something breaks, contact support and include the device model, iOS version, and a short description of the issue.

Contact

Email: support@timekitapp.com

App Store: Workbill on the App Store

Common questions

  • Premium purchases can be restored from the app settings screen using the same Apple ID that made the purchase.
  • Widget actions depend on App Groups being available for the app and widget on the same device.
  • Universal links for app events use the `gautam-u.github.io/workbill/event/...` path and should open Workbill on iPhone when the app is installed.
  • Work data remains on-device unless you choose Apple ecosystem sync features such as iCloud.
